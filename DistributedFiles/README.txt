# 概要
プロジェクト名はCommentAcquisitionEquipmentBot(長いから略してCAEBとでも呼んでね).
Twitchの配信からリアルタイムにコメントを取得することができます.
またコメント取得を発火に様々なアクションが発生します(現在は効果音と演出のみ).



# ショートカット一覧
* ctrl + ` = 設定を開く
* ctrl + q = アプリを閉じる
* ctrl + j = 配信にbotを参加させる
* ctrl + l = 配信からbotを離脱させる
* shift + ctrl + b = メニュウーバーの表示OnOff
* F11 = フルスクリーンOnOff



# 設定ファイル
アプリファイルのカレントディレクトリにDataBaseというフォルダが自動で生成されます.
その中に動画ファイルや音声ファイルを移動することをおすすめします.
settings.jsonは直接編集しないことをおすすめします.
## 演出
設定可能な動画ファイルは現在mp4のみとなっているため透過に対応しておりません.
透過したい背景はカラーコード#90EE90色に設定してください.
